# 数字逻辑校验

任何故事里出现钱、时间、债务、增长、成本，都要先过这张网。读者未必算账，但会感到“不对劲”。

## 基础公式

### 创业/开店

启动资金大致包含：

- 押金和首月租金
- 装修和设备
- 首批物料/库存
- 人员工资缓冲
- 平台推广/开业活动
- 至少 3-6 个月亏损缓冲

月固定成本大致包含：

- 房租
- 员工工资
- 水电物业
- 耗材/库存损耗
- 平台抽成/推广
- 贷款或信用卡利息
- 杂费、维修、退赔

退场残值：

- 装修通常很难回收
- 设备通常折价
- 库存可能积压
- 会员卡余额、押金、欠款、合同违约金会继续消耗现金

### 个人负债

债务形成要有时间线：

1. 先消耗存款
2. 再刷信用卡或分期
3. 再用小额贷款周转
4. 再找朋友/家人借
5. 最后变卖资产或增加工作时间

不要直接从“收入不错”跳到“欠几十万”。中间必须写明至少一个滚动机制：

- 收入不稳定但支出固定
- 最低还款导致本金没降
- 以贷养贷推高利息
- 合伙人退出导致缺口扩大
- 大额支出刚性化，短期无法止损

## 常用合理区间

以下不是事实，只是写作校验的生活常识区间。具体城市、行业、年份需要调整。

| 场景 | 合理区间 | 写作提示 |
|---|---:|---|
| 一线/新一线小店月租 | 1.5-8 万 | 商圈差异极大，郊区可低，核心商圈很高 |
| 小型门店启动资金 | 30-100 万 | 低于 30 万要解释为什么轻资产 |
| 普通门店人工 | 6000-12000/人/月 | 技师、销售、店长差异大 |
| 餐饮/美容/汽修月固定成本 | 5-20 万 | 房租和人工是核心 |
| 信用卡/消费贷债务滚到 | 10-50 万 | 需要 6-24 个月过程更可信 |
| 年轻人月薪 | 5000-20000 | 城市和行业必须匹配 |
| 每月稳定存款率 | 20%-70% | 超过 70% 要有住宿、食堂、低租金等条件 |
| 副业早期收入 | 0-5000/月 | 写上万要有渠道、技能或客户来源 |

## 常见漏洞

- “月入十万”但没有说明持续多久。一次性高收入不能支撑长期决策。
- “开店花了几十万”但没有拆成本。读者不知道钱去哪了。
- “流水十几万”直接等于赚钱。流水、毛利、净利必须区分。
- “欠债几十万”没有利息、周期和补窟窿动作。
- “卖车还债”但车没有贷款、折旧、残值逻辑。
- “合伙人退出”但没有说明股权、债务、转让关系。

## 写法建议

数字不确定时，不要编精确值。用区间或模糊精确：

- “一个月固定支出六七万”
- “前后投进去差不多六十万”
- “关店时还剩三十多万窟窿”
- “最开始还能靠存款撑，半年后就开始刷信用卡”

这种数字比“房租 23760 元、负债 382941 元”更像口述材料。
