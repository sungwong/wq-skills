# wq-gzh-ending：结尾生成

结尾的作用不是总结，是给读者一个**回味或动作**。别戛然而止，也别画蛇添足。

营销型结尾（转化CTA、加微信、报名）找 `/wq-gzh-marketing`，本skill只处理内容型结尾。

---

## 四种结尾模板

根据文章类型选择，或生成多个版本让用户挑：

### 1. 回扣开头型
把开头抛出的场景、问题或判断，在结尾给出呼应或答案。读者有"圆满了"的感觉。

适合：有明确开头钩子的文章
```
[呼应开头的场景/判断] + [一句更进一步的收束]
```

### 2. 引发思考型
抛出一个问题，让读者带着思考离开。不给答案，但问题本身有分量。

适合：观点类、认知类文章
```
[一个真正值得思考的问题，不是"你觉得呢"这种泛问]
```
注意：不要每篇都用提问结尾，会让人审美疲劳。

### 3. 金句收束型
用一句有力的判断或金句收尾，可被截图传播。

适合：方法论类、干货类文章
```
[一句话，有明确立场，能脱离文章单独存在]
```
可调用 `/wq-gzh-viewpoint` 打磨这句话。

### 4. 情感收束型
给读者一种温暖、被理解、或被鼓励的感觉。不说教，不煽情。

适合：共鸣类、成长类文章
```
[站在读者角度说一句真心话，不是"加油""相信你"这类空话]
```

---

## 禁止项

- ❌ 每篇都用提问结尾（单调）
- ❌ "希望这篇文章对你有帮助"（无意义）
- ❌ 重复正文已说过的内容当总结
- ❌ 强行拔高升华（"这不只是X，更是Y"）
- ❌ 硬插营销CTA（那是 `/wq-gzh-marketing` 的事）
