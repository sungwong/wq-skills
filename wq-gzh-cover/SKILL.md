# wq-gzh-cover：公众号封面图生成

---

## 生成通道（最高优先级）

只走浏览器 web 通道，禁止一切 API 和本地绘图：

1. **Gemini web**：CDP 连真实已登录 Chrome 的 Gemini 页面
2. **ChatGPT web**：CDP 连真实已登录 Chrome 的 ChatGPT 页面

禁止：任何 API（OpenAI / Google / DashScope / Replicate 等）、`baoyu-image-gen` API 模式、本地 PIL/脚本绘图、SVG 拼图。

通道不通时停下来告诉用户，不偷换通道。

生成后立即去水印，然后用 Preview 打开确认。

---

## 通用要求

- 生成前先把最终提示词保存到 `_workspace/cover-image/[slug]/prompts/`
- 生成后检查：比例、主体安全区、有无错字、符合账号识别
- 上传时 `--cover` 必须指向最终封面图

---

## 破局邦

- 先读最新排版规范：`02-方法论/AI指令库/写作指令/破局邦-排版规范.md`
- **比例**：2.35:1，推荐尺寸 `900×383`
- **风格**：无字新中式、水墨、纸影、剪纸、宣纸留白
- **不再固定黑马母题**：每篇根据文章主题选择具体意象；不要默认使用马或黑马
- **禁止放文字**：不放标题、不放"破局邦"、不放标签、不放二维码
- 生成前从文章提炼一个视觉场景或核心意象，保持账号气质统一，但画面要随主题变化
